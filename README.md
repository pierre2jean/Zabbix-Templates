# Zabbix-Templates #



## Installing Ruckus ZoneDirector Template ##



  Create two value maps under:  Administration => General => Value Mapping => Create ValueMap

  **RuckusAPStatus**

    0 ⇒ Disconnected
    1 ⇒ Connected
    2 ⇒ ApprovalPending
    3 ⇒ UpgradingFirmware
    4 ⇒ Provisioning


  **ruckusZDEthStatus**

    1 ⇒ Up
    2 ⇒ Down
